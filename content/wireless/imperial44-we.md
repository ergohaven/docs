Imperial44 Wireless Edition - беспроводная программируемая механическая сплит-клавиатура с хотсвап-системой для быстрой замены MX-совместимых свитчей и возможностью установить по одному хотсвап энкодеру на каждую половинку

## Устройство клавиатуры

![image](/images/imperial_we/imperial_we.png)

1. кнопка RESET
2. интегрированое кольцо MagSafe для УСН
3. силиконовые ножки
4. вспомогательное окно для разбора корпуса клавиатуры
5. переключатель питания клавиатуры
6. прозрачная панель
7. место для установки MX-свитча либо хотсвап-энкодера
8. USB-C разъем

## Первое подключение

- Клавиатура Imperial44 – Wireless Edition работает с устройствами под управлением Windows, macOS, Linux, Android, iOS
- Подключение клавиатуры осущевствляется посредством Plug and Play (PnP), дополнительная установка драйверов не требуется
- Клавиатура поставляется готовой для подключения к вашему устройству с актуальной на момент покупки прошивкой

{{% details title="Подключение напрямую (без Ergohaven's Qube)" closed="true" %}}
1. **Включите питание на левой и правой половинках клавиатуры** - половинки уже синхронизированы между собой
2. **Выберите профиль подключения на клавиатуре** - если нужно подключить на несколько устройств, для первого подключения можете пропустить этот шаг
3. **Подключите клавиатуру к устройству** - включите Bluetooth на устройстве, затем выполните сопряжение в настройках устройства
{{% /details %}}


{{% details title="Подключение через Ergohaven's Qube" closed="true" %}}
**Ergohaven's Qube** — универсальный донгл для беспроводных клавиатур Ergohaven, который отображает полезную информацию и снижает энергопотребление клавиатуры

1.  **Используя Type-C кабель подключите Ergohaven's Qube к компьютеру** и дождитесь загрузки Qube
2.  **Включите питание на левой половинке клавиатуры** - половинка автоматически подключится к Qube
3.  **Включите питание на правой половинке** - половинка также подключится к Qube, завершая настройку
{{% /details %}}

## Обновление прошивки

{{% details title="Прошивка для клавиатуры" closed="true" %}}
- Скачайте [reset прошивку](https://github.com/ergohaven/ergohaven-zmk/releases/download/2025.11.30/settings_reset-ergohaven-zmk.uf2) и актуальные прошивки для [левой](https://github.com/ergohaven/ergohaven-zmk/releases/download/2025.11.30/imperial44_left-ergohaven-zmk.uf2) и [правой](https://github.com/ergohaven/ergohaven-zmk/releases/download/2025.11.30/imperial44_right-ergohaven-zmk.uf2) половинок

- Выключите обе половинки переключателем с обратной стороны клавиатуры

1. Подключите USB кабель питания в левую половинку, дважды нажмите кнопку Reset с обратной стороны клавиатуры
2. Дождитесь когда контроллер перейдёт в режим загрузчика и откроется папка-хранилище (если на вашем устройстве отключен автозапуск подключаемых дисков, то перейдите в папку-хранилище вручную)
3. Скопируйте или перетащите Uf2 файл "reset прошивки", после чего контроллер должен моментально перезагрузиться. 
>  Reset прошивка нужна для затирания внутренней памяти клавиатуры. Она сбрасывает настройки профилей подключения и синхронизации половинок
4. Снова дважды нажмите кнопку Reset с обратной стороны клавиатуры и дождитесь когда контроллер перейдёт в режим загрузчика (откроется папка-хранилище)
5. Скопируйте или перетащите Uf2 файл прошивки для левой половинки, после чего контроллер должен моментально перезагрузиться

- Повторите 1-5 шаги для правой половинки

- Включите обе половинки переключателем с обратной стороны клавиатуры
- Одновременно зажмите на половинках кнопки Reset с обратной стороны клавиатуры и отожмите, это синхронизирует половинки между собой
- Подключите клавиатуру к вашему устройству
{{% /details %}}
